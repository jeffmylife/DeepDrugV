# DeepDrugV
DeepDrugV constructs the voronoi diagram (VD) of protein binding site/pocket or ligand structure based on the 3D  or 2D coordinate structure.

#Requirements
Python 2.7+
numpy 1.14+
scipy 0.18+
Pandas 0.19+
scikit-spatial 0.12.0
matplotlib 2.0.2+
biopandas ; eg : pip/conda install biopandas
